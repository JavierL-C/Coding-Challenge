# Coding-Challenge

Movie app written in Swift 4 with TMDb API and using app coordinator pattern, MVVM, UIKit, swiftUI.

Also this app include the following implementations:
- login view
- compositional layout
- Infiniti scroll
- cache image
- custom async image
- unit test
